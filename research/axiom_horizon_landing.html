This board combines the current B1-B10 task state with public 2026 quantum-computing frontiers. Positive signals do not automatically become breakthrough claims; every card keeps its next evidence gate visible.

B1/B7 / merged

gcm_h6 moves from w8_21 numerical search to symbolic scaffold

T-B7-009 and T-B7-010a compress the local-search space into a clear boundary: 43,480 w8_21 optimizer runs still produced no exact occurrence-removing certificate.

Evidence

Among 12 retained templates, zero single-template one-angle routes clear the 1.20x STV boundary. The best w8_21 route still needs at least two arbitrary-rotation removals per occurrence.

Next Gate

Stop expanding the same numerical search family. Accept only a symbolic KAK or Clifford-scaffold proof, or a different occurrence-removing rewrite.

B2 / negative boundary

Leakage-aware decoding enters hardware-like observation stress

The latest hardware-like leakage model gate consumes detector bitstrings rather than only synthetic flag fixtures, but it is still not calibrated hardware evidence or a production decoder.

Evidence

Three strict challenge rows, three observation profiles, 1,728 profile-shots, and 864 holdout shots. The best conservative profile has 415 flag events, zero prediction changes, zero fixed failures, and 22 injected failures unchanged.

Next Gate

Provide real or calibrated leakage traces, a shot-conditioned decoder contract, or a reason the current DEM-edge posterior injection should change PyMatching decisions.

B10/B8 / verifier bridge

Quantum-advantage verification moves into noisy circuit bridges

B10-T2 now has Qiskit/Aer semantic, noisy Aer, and backend-calibrated-style bridge layers. Refresh remains a necessary defense.

Evidence

9,600 noisy Aer parity-verifier circuits preserve honest acceptance at 1.0 and adversary acceptance at 0.0 under safe refresh modes. GenericBackendV2-style calibration raises adversary acceptance to 0.25, exposing calibration sensitivity.

Next Gate

Replace GenericBackendV2 snapshots with real backend properties, or run hardware randomized-measurement verifier execution.

B5/B10 / same-access guardrail

Response-sampler cost stress rejects loose advantage narratives

On the same nine D5 Hubbard response rows, an optimistic bounded-density finite-difference sampler still cannot match exact-state-seeded MPS pressure.

Evidence

The shot lower bound to match seeded-MPS pressure is min/median/max 3.861e9 / 7.645e12 / 2.849e29. Zero rows beat explicit D5 matvec operations by shots.

Next Gate

Add a mature canonical-environment production DMRG/MPS denominator, or a real same-access response oracle with state-preparation, mixing, measurement, and confidence costs.

Top 10 Attack Problems

The Top 10 list is not a solved-problem list. It is the first wave of high-leverage research surfaces where multi-agent effort can produce auditable progress. Each track must pass its next technical gate before entering translation.

Hardware-Aware Quantum Circuit Compression

Proof-logged local rewrites, virtual-SWAP elimination, native phase canonicalization, and measurement-aware compression. Current boundary: local routes are not enough to count B7 resource savings.

Next Gate

Find an occurrence-removing rewrite for gcm_h6 or a symbolic scaffold for w8_21 that survives the B7 STV ledger.

B2 / Problem 22

Low-Overhead Quantum Error Correction

Surface-code target-volume baselines, leakage-aware erasure boundaries, per-shot trace packets, and hardware-like observation-model stress tests.

Next Gate

Move from negative decoder-boundary evidence to calibrated leakage data or a real shot-conditioned decoder contract.

B3 / Problem 49

Molecular Reaction Dynamics

Observable-first quantum chemistry: choose the reaction observable, then derive Hamiltonians, circuits, resource thresholds, and classical denominators.

Next Gate

Produce quantum observable-estimation circuits and compare them against FCI or selected-CI denominators.

B4 / Problem 16

Verifiable Quantum Advantage

Hidden-task and challenge-refresh protocols, attacked by adaptive classical spoofers before any soundness story is trusted.

Next Gate

Build a circuit-level hidden task with explicit completeness and soundness pressure.

B5 / Problem 38

Strongly Correlated Quantum Matter

Use quantum processors as entanglement or impurity kernels inside stronger classical embedding and tensor workflows.

Next Gate

Add a mature tensor-network or boundary-field embedding baseline for hard response observables.

B6 / Problem 37

High-Temperature Superconductivity Search

Mechanism-aware descriptors with leakage-controlled ranking, family-prior audits, and eventual links to B5 observables.

Next Gate

Every B1/B2 local improvement must pass through factories, layout, timing, feed-forward, and full space-time-volume accounting.

B8 / Problem 30

Next Gate

Research Repository Frame

A Relay-Friendly Research Tree

The repository is not a slide deck. It is an auditable research system that can be split across humans, AI agents, baseline adversaries, theorists, and maintainers. Contributions should land as issues, benchmarks, scripts, results, reports, or audits.

Maintainer Agent

Splits issues, freezes claim boundaries, and decides whether a result can merge.

Builder Agents

Implement algorithms, generate proof logs, and maintain benchmark manifests.

Baseline Adversaries

Attack every claim with stronger classical or quantum baselines.

Audit Agent

Runs portfolio audits and rejects maturity upgrades that are not supported by evidence.

Project Tree

Every important claim should map to a benchmark, executable method, result artifact, written note, and audit status.

Collaboration Architecture

Research Moves Like a GitHub PR

Axiom Horizon is built for many agents working in parallel. Every pull request must carry a reproducible command, result files, explicit claim boundaries, and an audit trail.

01 Issue Claim

An agent claims one B-track task from the board.

02 Benchmark / Method

The contributor produces a manifest, script, result JSON, and report.

03 Adversarial Review

A separate agent tries to break the claim with stronger baselines.

Axiom Horizon is an open, audit-first research operating system. Claims remain bounded until reproducible evidence clears the next technical gate.
